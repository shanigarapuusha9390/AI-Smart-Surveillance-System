# AI-Smart-Surveillance-System
AI Smart Surveillance System is a real-time computer vision project that detects intrusion, fights, suspicious movement, and aggressive behavior from live video. It uses person detection, tracking, pose estimation, and facial analysis to trigger voice alerts, email notifications, and save incident clips for security monitoring.


## Features
- Intrusion detection in restricted zones
- Fight detection using pose estimation
- Suspicious movement detection
- Facial anger detection
- Real-time person tracking
- Email alert system
- Voice warning alerts
- Automatic incident clip saving
- Live monitoring dashboard

## Installation
git clone https://github.com/shanigarapuusha9390/AI-Smart-Surveillance-System.git
cd AI-Smart-Surveillance-System
pip install -r requirements.txt
python main.py
